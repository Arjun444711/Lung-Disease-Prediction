AI-Powered Lung Disease Detection

📌 Project Overview
This project uses Deep Learning to detect lung diseases from chest X-ray images.

🚀 Features
- X-ray image upload
- Disease prediction
- Deep Learning model
- User-friendly interface

🛠 Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Pandas
- Streamlit

📂 Dataset
Dataset: NIH Chest X-ray Dataset
Link: https://www.kaggle.com/datasets/nih-chest-xrays/data
